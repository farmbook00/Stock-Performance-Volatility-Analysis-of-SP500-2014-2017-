# Stock-Performance-Volatility-Analysis-of-SP500-2014-2017-
The objective of this project is to analyze historical stock performance, volatility, and trading activity of S&amp;P 500 companies between 2014 and 2017 to identify trends, risk patterns, and high-performing assets.
